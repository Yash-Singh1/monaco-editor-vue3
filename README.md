# `monaco-editor-vue3`

The Monaco Editor for VueJS v3.

## Install

```sh
npm i monaco-editor-vue3
```

## Usage

```vue
<script setup>
import MonacoEditor from 'monaco-editor-vue3';
</script>

<template>
  <MonacoEditor onChange="onChangeMethod" initialValue="console.log('Hello World!')" />
</template>
```
